# dzika-spizarnia-product-case -MPV

Dzika Spiżarnia to mobilna aplikacja webowa stworzona jako MVP dla osób zainteresowanych ziołami, naturalnymi metodami dbania o zdrowie oraz sezonowością zbiorów.

Celem projektu było uporządkowanie rozproszonej wiedzy i umożliwienie użytkownikowi szybkiego znalezienia:
- zastosowania zioła
- przepisu (forma użycia)
- informacji o sezonie zbiorów

Użytkownik:
- chce szybko sprawdzić „na co działa” dane zioło
- szuka konkretnej formy (napar, ocet, maść)
- nie chce przeszukiwać wielu źródeł
- potrzebuje prostego i intuicyjnego interfejsu mobilnego

Pierwsza wersja aplikacji obejmuje:

- wyszukiwarkę wielopolową:
  - „Na co?”
  - „Z czego?”
  - „Forma”
- listę wyników jako czytelne karty
- szczegóły przepisu:
  - opis działania
  - składniki
  - kroki
  - przeciwwskazania
- kalendarz sezonowości zbiorów
- formularz „Masz uwagi?” (mechanizm zbierania feedbacku)

Projekt został zaprojektowany jako:

- Mobile-first
- Minimalistyczny interfejs
- Jasny flow użytkownika:  
  Wejście → Wyszukiwanie → Wynik → Szczegóły → Powrót / Nowe wyszukiwanie

MVP umożliwia testowanie:

- czy użytkownicy częściej wyszukują po problemie („na co?”) czy po składniku
- które treści są najczęściej otwierane
- czy użytkownicy zostawiają feedback



Aplikacja posiada panel administracyjny umożliwiający:

- analizę lejka użytkownika (welcome → wyszukiwanie → wyniki → przepis)
- monitorowanie konwersji do przepisu
- identyfikację zapytań bez wyników
- analizę najczęściej wyszukiwanych haseł
